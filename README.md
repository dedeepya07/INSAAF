# INSAAF
INSAAF (Interactive System for National Adjudication and Assistance Framework) is an AI-powered platform transforming legal assistance in India. Leveraging Llama API for multilingual NLP, HostGator’s scalable cloud, and robust security (AES, SSL, MFA), it offers a smart chatbot, case tracking, fine payments, and eCourts integration.
